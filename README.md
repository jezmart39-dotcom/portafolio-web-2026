🛠️ Tecnologías y Herramientas

Este proyecto fue construido desde cero aplicando separación de responsabilidades y buenas prácticas (cero código comentado y correcta indentación):

HTML5: Estructura semántica.
CSS3: Flexbox, CSS Grid, variables CSS, animaciones personalizadas (keyframes) y diseño responsivo (Dark/Light Mode).
JavaScript (Vanilla): Modularizado en script.js para manipulación del DOM, Intersection Observer (animaciones al hacer scroll) y lógica del carrusel interactivo.
Git & GitHub: Control de versiones.
Despliegue: Netlify CI/CD.

✨ Características Principales

Modo Claro / Oscuro: Implementación fluida que adapta tanto la paleta de colores como la luminosidad de los assets gráficos.
Animaciones Pop-Up & Reveal: Micro-interacciones suaves que mejoran la experiencia de usuario.
Bento Grid: Interfaz moderna para la sección de habilidades y especificaciones técnicas.
Carruseles Dinámicos: Carrusel infinito para tecnologías y carrusel manual para visualización de entornos (Xcode/iOS).
